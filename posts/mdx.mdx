title: Об использовании Markdown для создания блогов
description: Чем это удобно
public: true
date: '2020-10-18T19:50'
---

Markdown намного удобнее для написания блогов, чем, например, HTML. Это формат, созданный
именно для написания текста, а не для его форматирования. Если вам не нужно какое-то
особенное форматирование, Markdown выглядит как обычный текст.

В то же время, он предоставляет богатые возможности по добавлению более сложного контента,
а также может поддерживать метаданные в формате YAML с помощью front matter.
